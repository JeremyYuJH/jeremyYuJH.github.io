# Introduce

这是基于 Vue.js 实现的移动端组件库

# 愿景

我们希望这是一个可以兼顾易用性和灵活性的组件库。

# 特点

  • 拓展性强，适用多种场景。
  • 支持按需引用

# 安装

npm  install light-ui-m --save
